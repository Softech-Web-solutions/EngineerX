What is a Karnaugh Map?

A Karnaugh Map (K-map) is a graphical tool used to simplify Boolean algebraic expressions. It provides a systematic method for minimizing Boolean functions by visualizing the relationships between different variable combinations.

Key Concepts:

  • Minterms (SOP): Sum of Products form - represents combinations where the function outputs 1
  • Maxterms (POS): Product of Sums form - represents combinations where the function outputs 0
  • Don't Care: Conditions that can be either 0 or 1, used for further optimization
  • Grouping: Adjacent cells with same values can be grouped to simplify expressions

How to Use:

  • Select the number of variables (2, 3, or 4)
  • Choose between Minterms or Maxterms input
  • Enter your terms as comma-separated numbers
  • Click Generate to see the K-Map and simplified expression
  • Click on cells to toggle values manually

Grouping Rules:

  • Groups must contain 1, 2, 4, 8, or 16 cells (powers of 2)
  • Groups can wrap around edges of the K-map
  • Larger groups result in simpler expressions
  • Each group eliminates variables that change within the group
